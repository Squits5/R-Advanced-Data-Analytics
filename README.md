# R-Advanced-Data-Analytics

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![R Version](https://img.shields.io/badge/R-4.0%2B-blue.svg)](https://www.r-project.org/)

Collection of advanced data analysis and visualization scripts using R. This repository focuses on demonstrating statistical modeling, machine learning workflows, and creating insightful interactive reports for complex datasets.

## Features

-   **Exploratory Data Analysis (EDA):** Scripts for comprehensive data understanding.
-   **Statistical Modeling:** Examples of linear regression, logistic regression, and time series analysis.
-   **Machine Learning:** Integration with `caret` for model training and evaluation.
-   **High-Quality Visualizations:** `ggplot2` for static and interactive plots.

## Installation & Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Squits5/R-Advanced-Data-Analytics.git
    cd R-Advanced-Data-Analytics
    ```

2.  **Install R packages:**
    Open R and run:
    ```R
    install.packages(c("ggplot2", "dplyr", "tidyr", "caret"))
    ```

3.  **Run the analysis script:**
    ```bash
    Rscript src/data_analysis.R
    ```

    This will execute the analysis and save plots to the current directory.
